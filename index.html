Arsip & Cadangan Google Drive

Simpan atau pulihkan seluruh database pegawai beserta status unggahan berkas ke Google Drive Anda dalam format JSON.

Tips Hemat Penyimpanan (Firestore Safe): Kami sangat menyarankan menggunakan opsi "Tautkan Link G-Drive" untuk melampirkan berkas dokumen. Metode tautan link ini tidak menggunakan ruang penyimpanan cloud database (hanya memakan beberapa byte), sehingga aman dari batas penyimpanan maksimal database dan menghindari kegagalan sinkronisasi berkas berukuran besar.
